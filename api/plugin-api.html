FigML Plugin API Reference

This document provides a comprehensive reference for the FigML Plugin API, allowing you to integrate FigML's functionality into your own workflows.

Core API Methods​

convertToXml(options)​

Converts Figma designs to XML format.

Parameters​

ParameterTypeDescription
optionsObjectConfiguration options for the conversion
options.selectionBooleanIf true, only converts selected elements
options.includeStylesBooleanIf true, includes style information in the output

Returns​

{
  xml: string;       // The generated XML
  statistics: {
    elements: number;  // Number of elements converted
    depth: number;     // Maximum nesting depth
  }
}

Example​

const result = figml.convertToXml({
  selection: true,
  includeStyles: true
});

console.log(result.xml);

parseXml(xmlString)​

Parses XML back into Figma elements.

Parameters​

ParameterTypeDescription
xmlStringStringThe XML string to parse

Returns​

A Figma node object that can be added to the document.

Events​

onConversionComplete​

Triggered when a conversion operation completes.

figml.onConversionComplete((result) => {
  console.log('Conversion completed!', result);
});

Error Handling​

All API methods can throw exceptions when errors occur. It's recommended to use try/catch blocks:

try {
  const result = figml.convertToXml({selection: true});
} catch (error) {
  console.error('Conversion failed:', error.message);
}

Configuration​

The FigML plugin can be configured using the configure method:

figml.configure({
  indentation: 2,
  includeComments: true,
  namingStrategy: 'camelCase'
});

For more detailed examples and use cases, refer to our Examples section.
