Installing FigML

Welcome to the FigML installation guide. This document will walk you through the steps to get FigML up and running.

Prerequisites​

Before installing FigML, make sure you have:

  • A Figma account
  • Basic understanding of XML
  • Node.js v14 or higher (if using the CLI)

Installation Options​

Option 1: Figma Plugin​

  1. Open Figma and navigate to the Community tab
  2. Search for "FigML"
  3. Click Install
  4. The plugin will now be available in your Figma plugins list

Option 2: CLI Tool​

npm install -g figml-cli

Verifying Installation​

To verify that FigML has been installed correctly:

  1. For the Figma plugin: Open Figma and check if FigML appears in your plugins list
  2. For the CLI tool: Run figml --version in your terminal

Next Steps​

Now that you have FigML installed, check out our Quick Start Guide to begin using FigML.
